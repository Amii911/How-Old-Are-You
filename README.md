# How Old Are You Calculator

A simple Python project that calculates a user's age in months.

## Overview

This program asks the user for their age in years and converts it into the total number of months they have lived.

The calculation uses a simple formula:

```python
months = age * 12
```

Since there are 12 months in one year, multiplying the age by 12 gives the user's age in months.

## Features

- Accepts user input
- Converts years into months
- Displays results in a readable format

## Technologies Used

- Python 3

## Installation

1. Clone this repository:

```bash
git clone https://github.com/yourusername/How-Old-Are-You.git
```

2. Navigate to the project folder:

```bash
cd age-in-months
```

3. Run the program:

```bash
python app.py
```

## Usage

Enter your age when prompted:

```text
Enter your age in years: 25
```

Output:

```text
You are 300 months old.
```
